# Kotlin-Angular2
Demo app based on Kotlin and Angular 2

## Setup
```cd``` into angular app

```
npm install
npm start
```
---------------------------------

```cd``` into kotlin app
<br/>
&nbsp;***ensure you have maven***
<br/>
 &nbsp;***ensure your JAVA_HOME is directed at JDK***
<br/>
<br/>
&nbsp;run ```mvn spring-boot:run```

&nbsp;Goto: http://localhost:4200
